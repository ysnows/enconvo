# Enconvo 2.5.5 Beta 26

Thanks for testing Enconvo 2.5.5 Beta 26. This build focuses on making account
usage clearer, improving Inline Mode reliability, and polishing recording,
captioning, transcription, and result surfaces.

## Highlights

- **Points usage is easier to understand** with clearer allowance reset history
  and larger Cloud top-up packs at the same prices.
- **Inline Mode is more reliable in Mail** and now places its progress window
  next to the pointer on the correct display.
- **Recording, captioning, and transcription pages feel cleaner** with better
  title spacing, compact toolbars, and improved long-title handling.
- **Recording and live-caption mentions are more dependable** because they use
  the correct recording attachment identity and clean up stale mentions.
- **Completion prompts are less noisy** when the same conversation is already
  visible in another Enconvo surface.

## Points Usage

- Fixed monthly allowance resets so points history shows the resulting
  allowance, such as `Reset to 500,000`, instead of only showing a partial
  credit difference.
- Changed Cloud top-up packs to include more points at the same prices: `$10`
  grants `450,000` points, `$50` grants `2,250,000`, and `$100` grants
  `4,500,000`.

## Inline Mode

- Fixed PopBar Inline Mode in Mail so a shortcut selects and replaces the
  editable message body when no text is already selected.
- Fixed Inline Mode progress placement so the progress window appears next to
  the pointer on the correct display.
- Fixed the Chat Window so pressing Return sends the current prompt without
  changing Return behavior in other windows.

## Recording, Captioning, And Transcription

- Changed speech model page titles to sit more comfortably below the macOS
  window controls.
- Fixed recording, captioning, and transcription detail pages so long titles no
  longer overlap the window controls or cause horizontal scrolling.
- Changed recording, captioning, and transcription lists to use a compact
  title-free toolbar.
- Changed live captioning and recording to default live translations to
  Japanese, migrate the previous Chinese default on existing installs, and hide
  the setup panel's scroll indicator.
- Fixed recording and live-caption mentions so they use the recording
  attachment ID, include the transcript directly, and remove stale mentions
  when recordings are deleted or restored.

## Results And Session Completion

- Changed native visual result cards to use tighter outer padding and clearer
  spacing around colored callouts.
- Fixed completed sessions so they no longer reopen Dynamic Island or send a
  notification while the same conversation is already visible in another
  Enconvo surface.
