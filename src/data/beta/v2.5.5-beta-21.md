# Enconvo 2.5.5 Beta 21

Thanks for testing Enconvo 2.5.5 Beta 21. This build focuses on clearer plan
choices, better Cloud model defaults, more predictable PopBar behavior, and a
smoother setup path for Blender workflows.

## Highlights

- **Teams lifetime licenses are available** with seat-counted device limits,
  bonus points per seat, and lifetime updates.
- **Cloud plans are easier to compare and upgrade** with richer plan cards,
  full benefits tables, clearer pricing links, and prorated one-click upgrades.
- **Model options are stronger** with Kimi K3, MiniMax M3, improved LM Studio
  metadata, and MiniMax M3 as the default Enconvo Cloud Plan model.
- **PopBar is less likely to appear at the wrong time** after window drags,
  title-bar double-clicks, or object clicks in design apps.
- **Blender setup is easier** with the community blender-mcp preset and a
  one-click add-on download.

## Plans And Billing

- Added a Teams lifetime license with seat-counted device limits, 50,000 bonus
  points per seat, and lifetime updates.
- Added a Teams card with a live seat selector to the in-app plan picker and
  website pricing page, including seat add-on purchases for existing Teams
  accounts.
- Added Teams details to the Account page so Teams users can see seat count and
  lifetime update status.
- Changed device limits to follow account entitlements everywhere: Standard,
  Pro, Teams, and Cloud now use their own limits during login and app use.
- Added grouped benefits comparison tables to the in-app plan picker and website
  pricing, including one-free-session trials for meeting recording and live
  captions.
- Changed Cloud plan cards to list their complete entitlements, and moved the
  Free tier into the website comparison table.
- Added Enconvo Cloud tiers Plus, Pro, and Max to the plan picker and Account
  page, including bundled monthly points and discounted DeepSeek / MiniMax M3
  usage.
- Added one-click Cloud tier upgrades in the plan picker, with prorated charges
  shown before confirming the upgrade.
- Added a pricing link to the Enconvo Cloud Plan provider settings.
- Changed Cloud pricing links so Account settings opens detailed model and
  service rates while View Cloud plans returns to the website plan selector.
- Changed the Account page to show "Ends <date> · won't renew" for cancelled
  Cloud plans instead of a next billing date.

## Models

- Added Kimi K3 as a Cloud plan model and as a selectable model for your own
  Kimi API key, with its 1M-token context and always-on maximum reasoning.
- Added MiniMax M3 with 512K context for both own-key and Enconvo Cloud use, and
  made it the default MiniMax model.
- Changed the Enconvo Cloud Plan default model to MiniMax M3.
- Changed the LM Studio model list to use the LM Studio REST API, so vision,
  reasoning, and tool-use capabilities are detected automatically and each model
  shows its real display name and context window.

## PopBar

- Fixed PopBar appearing after moving or resizing a window by recognizing window
  drags, including background windows and apps with limited accessibility
  support.
- Fixed PopBar showing a previously selected value when double-clicking a
  window title bar to zoom the window.
- Fixed PopBar appearing when double-clicking a layer or canvas object in design
  apps by skipping recognized non-text elements and discarding copied object
  content.

## App Presets And Website

- Changed the Blender App Sidebar preset to the community blender-mcp project at
  blendermcp.org, with updated add-on install and connection steps.
- Added a one-click "Download addon.py" button to the Blender setup card that
  saves the add-on to Downloads and reveals it in Finder.
- Changed the enconvo.com homepage to the new positioning, tabbed hero showcase,
  model-freedom and open-platform sections, Pet/IM sections, and pricing groups
  for bring-your-own-AI and Enconvo Cloud plans.
- Fixed the website Cloud pricing cards showing a blank white buy button on the
  highlighted plan, and aligned the website pricing layout with the in-app plan
  picker.
