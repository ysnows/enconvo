# Enconvo 2.5.5 Beta 22

Thanks for testing Enconvo 2.5.5 Beta 22. This build focuses on safer in-app
purchases and more reliable PopBar behavior in apps with incomplete
accessibility metadata.

## Highlights

- **In-app purchases are more reliable** because checkout requests now go
  through the API worker.
- **PopBar behaves better in Sketch and similar apps** when accessibility does
  not expose text metadata.

## Billing

- Fixed in-app purchases failing when the website is behind a security
  checkpoint by creating checkout sessions through the API worker.

## PopBar

- Fixed PopBar text-element detection in apps such as Sketch by allowing the
  current selection flow to continue when accessibility text metadata is empty.
