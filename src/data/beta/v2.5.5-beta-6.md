# Enconvo 2.5.5 Beta 6

Thanks for testing Enconvo 2.5.5 Beta 6. This build focuses on Projects,
clearer code output, xAI search reliability, and a smoother Voice Command
recording experience.

## Highlights

- **Projects are now supported in Chat Window**, letting you group related
  chats under a folder-backed workspace with project instructions and project
  context.
- **Code blocks now support syntax highlighting**, with a cleaner light and
  dark code theme for chat replies, tool results, logs, and file previews.
- **xAI search is fixed**, resolving the xAI X/Twitter web search error on the
  Enconvo Cloud plan.
- **Voice Command audio cues are smoother**, with improved prompt sound and
  mute-restore behavior around recording start, stop, and cancel flows.

## Projects

- Added Projects to the Chat Window sidebar so related chats can be grouped
  under a folder-backed project.
- Added project creation from the sidebar using either an existing folder or a
  new folder created under `~/Documents`.
- Added a Project Instructions editor that saves to the project's `AGENTS.md`
  file and applies those instructions to every chat in the project.
- Added project actions for Reveal in Finder, Rename project, Project
  Instructions, Delete chats, and Remove; removing a project keeps its chats and
  never touches the folder on disk.
- Changed project agent chats to run from the project folder and follow project
  instructions from Chat Window, SmartBar, and scheduled runs.
- Added a project chip under the chat input for new chats, with search, quick
  project creation, and a sticky default that remembers the last selected
  project.
- Added a git branch chip for project folders that are git repositories,
  showing the current branch and uncommitted file count with branch switching
  and create-and-checkout support.
- Changed new project folders to initialize as git repositories automatically
  when git is installed.

## Code Output

- Fixed code blocks in chat rendering without syntax highlighting.
- Changed chat code blocks to use a flat single-surface style with the language
  label and copy button on the same card.
- Fixed the chat message list not scrolling when the pointer is over a code
  block, matching the existing table and chart scroll behavior.

## Voice Command

- Fixed Voice Command sometimes leaving system volume muted after recording ends
  when "Mute while recording" is enabled.
- Changed Voice Command mute restoration so canceling a session, stopping right
  after the start sound, and multi-window sessions all restore the original
  volume correctly.

## Search And Provider Fixes

- Fixed xAI X/Twitter web search failing with a 404 on the Enconvo Cloud plan
  while direct xAI OAuth continued to work.
- Fixed the model selector ignoring clicks on other providers; picking a
  provider now always switches the model column to that provider's models.
- Fixed untitled leftover agents showing up as blank "AI Language Model Chat"
  entries in agent pickers.

## Points And Billing

- Changed the Points settings page to show your balance split into top-up
  points that never expire and subscription points that reset each cycle, each
  with a short explanation of whether it resets.
- Removed the old per-row Balance column from the usage history, replacing it
  with a per-entry breakdown of how much was spent from top-up versus
  subscription points.
- Changed purchased top-up points and Standard / Premium plan points to never
  expire, while Monthly / Yearly subscription points reset to the plan amount on
  each renewal.

## Workflow And API Reliability

- Fixed ten plugin API crashes on missing parameters, so affected endpoints now
  return clear errors instead of server crashes.
- Fixed adding text to a knowledge base without specifying one crashing on a
  database constraint; it now falls back to the default knowledge base.
- Fixed the OCR command crashing when called without an image; it now reports
  that `image_files` is required.
- Changed workflow templates to render objects and arrays as JSON
  automatically, eliminating silent `[object Object]` output when interpolating
  step results.
- Fixed hyphenated workflow step ids silently rendering `NaN` in templates by
  requiring snake_case identifiers for job and step ids.
- Removed the legacy flat `tasks:` workflow format; workflows now require
  `jobs` of sequential `steps`, with a clear migration message for old files.

## Chat Window Polish

- Changed the new chat screen to center the input like the welcome layout, with
  the model selector moved next to the send button.
- Changed the sticky new-chat project to follow the chat you open: opening a
  project chat targets that project, and opening a top-level chat clears it.
- Changed creating a project to immediately select it for the next chat, with
  new project folders starting on a main branch.
- Changed the sidebar Plugins entry to open the plugin browser directly in the
  chat area instead of jumping to the settings window.
- Changed the sidebar Automations entry to open the scheduled-jobs manager
  directly in the chat area instead of jumping to the settings window.
- Fixed dialogs flying in from the top-left corner; they now use the standard
  centered fade-and-zoom animation.
