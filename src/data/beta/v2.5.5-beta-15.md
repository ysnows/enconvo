# Enconvo 2.5.5 Beta 15

Thanks for testing Enconvo 2.5.5 Beta 15. This build adds a quick help menu to
the chat window so setup links and reference material are one click away.

## Highlights

- **A new help menu in the chat window footer** gathers the Chrome companion
  setup, release notes, and keyboard shortcuts into one place.

## Chat Window

- Added a help button to the bottom-right of the chat window that opens a menu
  to set up the Chrome companion extension, see what's new, and open the
  keyboard shortcuts settings.

## App Sidebar

- Changed background App Sidebars to stay stacked directly above their own
  app's window, so a sidebar remains visible next to its app while the
  Settings or Chat window is open instead of disappearing behind everything.
- Fixed a hidden app's sidebar briefly flashing on screen when the Settings
  or Chat window opens; fully covered sidebars now stay invisible through the
  transition.
- Fixed the collapsed sidebar toggle button sometimes appearing at the same
  time as the open sidebar when the panel opened during the toggle's debounced
  show.

## Smart Bar

- Fixed chat text in the Smart Bar being un-selectable with the mouse: the
  whole panel surface acted as a window-drag area, so drag-to-select moved the
  window instead; while a conversation is shown, dragging now only moves the
  window from the top toolbar and text selects normally.

## Recording

- Added a polite meeting-record reminder: when a Zoom, Microsoft Teams, or
  FaceTime call goes live, the Dynamic Island pops a semi-expanded pill with
  the app's icon asking whether to start recording — one tap opens the
  meeting-record flow, shown once when the meeting starts, with a Recordings-
  page toggle to turn automatic reminders off.
- Changed the Dynamic Island to not expand on hover while a recording or
  captioning session is running, so passing the pointer by never covers the
  live captions.

## AI Providers

- Fixed OpenAI ChatGPT (OAuth) chats failing with "Connection error." on
  GPT-5.6 Luna: the Codex backend now gates newer models by client version,
  so requests carry a client version and Luna works again.
- Changed OpenAI ChatGPT (OAuth) request failures to show the real error from
  OpenAI (for example "Model not found") instead of a generic
  "Connection error.".
- Fixed duplicating an AI model provider adding a second "Credential Provider"
  row to the duplicate's settings.
- Changed duplicated providers to own their auto-created credential: the
  credential now shares the provider's exact name, follows provider renames,
  and is deleted with the provider when nothing else uses it.
- Added identity subtitles to credential pickers and the Credentials settings
  page — OAuth credentials show the connected account email and plan, API-key
  credentials show a masked key tail — so duplicates are distinguishable.
- Changed the Credential Provider dropdown to pin the provider's own
  credential on top and group the rest under "Other credentials" with their
  owning provider labeled.
- Added "in use" and "Unused" indicators to the Credentials settings page,
  with a one-click cleanup for credentials no provider uses anymore.
- Added a one-time cleanup that removes leftover empty credential copies from
  earlier provider duplications and renames surviving ones to match their
  provider; credentials that still store a key are kept and marked "Unused"
  instead of being deleted.

## Website

- Added a Use Cases page to enconvo.com with video walkthroughs of real
  scenarios (Excel automation, Gmail workflows, provider setup, OCR, Agent
  Mode), category filters, a lightbox player, and shareable per-video links,
  plus Use Cases links in the site navigation and footer.
- Fixed the enconvo.com changelog page cutting off wrapped release note lines
  mid-sentence.
- Added release search, a scroll-synced release index with a mobile jump menu,
  and change-type badges to the enconvo.com changelog page.
- Changed the enconvo.com changelog page to show each release's intro summary
  and stop duplicating highlight entries in the section list.

## Screen Doodle

- Fixed Screen Doodle closing itself about a second after opening via the
  hotkey or markup button whenever the Double Right-Click gesture was turned
  off; the gesture toggle now only disables the right-click shortcut.

## Dynamic Island

- Fixed the Dynamic Island home layout shifting sideways while typing: the
  model pill no longer forces the toolbar wider than the island (long model
  names now truncate, capped in the island), so the input card and shortcut
  grid stay perfectly still as the Send button appears.
- Fixed "Disable Dynamic Island" in the menu bar menu: the island briefly
  vanished and then reappeared because a deferred layout refresh re-showed the
  closed panel; disabling now keeps it hidden until re-enabled.
- Fixed the Voice Input button being hard to click in the Smart Bar, App
  Sidebar, and Dynamic Island: only the thin waveform strokes were clickable,
  so most taps landed on dead space — the whole button now starts dictation.
- Fixed dragging or pasting a file from another app into the Dynamic Island
  deadlocking with the system "access data from other apps" permission prompt,
  where both the island and the prompt became unclickable; the island now
  yields focus so the prompt stays clickable.
- Fixed wide math formulas in chat replies being cut off at the Dynamic
  Island's edge with no way to see the rest; formula blocks now scroll
  horizontally, and vertical swipes over them keep scrolling the conversation.
