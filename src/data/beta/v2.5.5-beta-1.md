# Enconvo 2.5.5 Beta 1

Hi friends, this beta focuses on making visual answers feel native inside the
conversation. Enconvo can now show richer chat output directly where you are
working, with smoother widget streaming and a more reliable beta app startup.

## Highlights

- **Visual answers now appear directly in chat** with SVG, HTML previews, and
  interactive widgets.
- **Widgets feel smoother while they stream and update**, with better scrolling
  and fewer unwanted resets.
- **Beta startup is more reliable**, so this build should behave closer to a
  normal in-app update.

## Visual Answers

- Added a Visualize tool for rendering SVG, HTML, and sandboxed widgets inside
  chat responses.
- Widgets can stream into place and follow the current light or dark theme.

## Widget Reliability

- Scrolling over a widget now continues the chat scroll instead of trapping the
  page inside the widget.
- Widget updates are coalesced, so interactive state is less likely to reset
  while an answer is still streaming.

## Beta App Startup

- Release startup now prepares the bundled app resources needed by the beta
  build.
