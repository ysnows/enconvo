# Enconvo 2.5.5 Beta 25

Thanks for testing Enconvo 2.5.5 Beta 25. This build focuses on making Voice
Command appear instantly, keeping the Dynamic Island HUD visually stable, and
making Dictation translation behave consistently across models.

## Highlights

- **Voice Command opens much faster** after pressing the hotkey, so the HUD is
  ready almost immediately.
- **The live voice HUD looks cleaner** with centered transcript text, a subtle
  voice-wave cue, and no thin top-edge line on notched displays.
- **App switching is smoother** because accessibility health checks no longer
  block the main interaction path.
- **Dictation translation is more consistent** across realtime and
  post-action models.

## Voice Command

- Changed Voice Command startup so the HUD can appear within about 90 ms after
  the hotkey press.
- Fixed the live transcript layout so dictated text stays centered on the same
  line as the waveform.
- Added a subtle voice-wave glyph beside the Standing by state to make the idle
  voice action easier to recognize.

## Dynamic Island HUD

- Fixed a thin white line that could appear along the top edge of notched
  displays while the voice HUD was open.
- Improved the visual balance of the Standing by pill so it feels less empty
  before recording starts.

## Responsiveness And Dictation

- Fixed multi-second stalls when switching apps by moving accessibility server
  checks out of the main interaction path.
- Changed Dictation translation so realtime models and post-action translation
  models now apply translation through the same final step.
