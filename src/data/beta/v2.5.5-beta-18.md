# Enconvo 2.5.5 Beta 18

Thanks for testing Enconvo 2.5.5 Beta 18. This build focuses on making side
panels, chat steering, workflow runs, and app-specific editing feel more direct
and dependable.

## Highlights

- **App Sidebar feels much smoother** when it opens, follows app windows, closes,
  and restores the original window position.
- **Chat steering and queued messages are more reliable** across Chat Window,
  SmartBar, Dynamic Island, App Sidebar, and IM channels.
- **Workflow runs are easier to follow and stop** with live visual run canvases,
  proper cancellation, and clearer step output.
- **Office and Finder skills are now more practical** through typed operations
  for spreadsheets, documents, slides, files, and Finder selections.
- **Knowledge Base imports are faster and more useful** for PDFs, images, and
  large libraries.

## App Sidebar

- Changed App Sidebar opening and closing to feel instant, including faster
  collapsed-handle recovery and no leftover fade delay.
- Fixed App Sidebar window restoration so the app returns to its original size
  and position after the sidebar closes, unless you moved it yourself.
- Changed App Sidebar layout so the host app keeps its full size whenever the
  app and sidebar can fit together, shrinking only when necessary.
- Fixed App Sidebar dragging and window tracking so the sidebar follows the host
  window smoothly instead of lagging, resizing, or pushing during the drag.
- Changed App Sidebar window ordering so sidebars come forward and sink behind
  other apps together with their host app.
- Changed the App Sidebar empty state to center the app icon, title, and
  suggested prompts.

## Chat And Agent Control

- Added queued-message cards with edit, delete, reorder, and Steer actions,
  replacing the old floating queued-message overlay.
- Added Steer as an immediate course-correction: it interrupts the current
  response and starts the new instruction right away while preserving the rest
  of the queue.
- Added Cmd+Enter as the Steer shortcut across Chat Window, SmartBar, Dynamic
  Island, and App Sidebar.
- Fixed stopping and steering so the real worker stops, including background
  generation, running tools, and billing-sensitive work.
- Fixed chat replies sometimes staying stuck on "Thinking..." and never showing
  the streamed response.
- Fixed duplicate steered messages and phantom queued-message flashes when a run
  finishes at the same moment a message is sent.

## Workflow Runs

- Added a Stop button for workflow runs so pending steps are skipped and the run
  ends as cancelled right away.
- Added live workflow canvases to SmartBar workflow runs and agent chat, showing
  running steps, animated connectors, and click-to-view outputs.
- Added Open in Editor shortcuts from live workflow runs so an in-progress run
  can be inspected in the full editor.
- Fixed cancelled workflow runs leaving in-flight steps spinning forever in the
  run view.
- Changed agent inline batches to run independent work as a parallel workflow,
  with bounded waiting, retry/resume behavior, and trimmed long outputs.
- Fixed shell step cancellation so the full process tree is terminated instead
  of leaving orphaned child processes running.

## Office And Finder Skills

- Changed Excel, Numbers, Keynote, Pages, Word, and PowerPoint live editing to
  use packaged typed methods instead of AI-written AppleScript for common edits.
- Added office methods for finding content, navigating to locations, reloading
  from disk, formatting, inserting structure, editing charts, and returning the
  previous content for review.
- Added chart creation support for Excel and Keynote, with other office apps
  routing chart requests to file-based document workflows when live charting is
  not supported.
- Added a Finder skill for moving, copying, renaming, trashing, duplicating,
  compressing, tagging, revealing, selecting, and reading Finder selections.
- Added Finder image utilities for resizing, converting, and rotating images
  through the system image processor.
- Changed Apple Mail, Reminders, Calendar, Contacts, Messages, Notes, and Maps
  automation to live under the Application plugin, so App Sidebar
  recommendations can resolve and install them from one place.

## Knowledge Base And Files

- Changed PDF parsing to use embedded text directly for text-based PDFs, making
  imports faster and more accurate while keeping OCR fallback for scanned pages.
- Changed PDF parsing to extract embedded figures and reference them in Markdown
  so Knowledge Base and vision models can use the images.
- Fixed local OCR for HEIC and HEIF images by decoding them through the native
  macOS image converter when needed.
- Changed Knowledge Base image grids to use cached cover thumbnails instead of
  full-resolution originals for smoother browsing.
- Changed Knowledge Base switching to load the folder tree before the item grid
  so the sidebar no longer lags behind the content.
- Added per-library actions in the Knowledge Base switcher for renaming,
  reindexing, and deleting a knowledge base.

## Fixes

- Fixed the Copy result action after PopBar commands like Translate so it copies
  the command result instead of the originally selected text.
- Fixed MCP tool calls sending internal runtime fields to strict MCP servers.
- Fixed stopped agent sessions leaving inline workflow runs active in the
  background.
