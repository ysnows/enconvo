# Enconvo 2.5.5 Beta 4

Thanks for testing Enconvo 2.5.5 Beta 4. This build focuses on the details that
make Enconvo feel steadier in daily use: App Sidebar should stay where you
expect it, Dynamic Island input should behave more reliably, default agents
should be easier to switch, chat should be easier to interrupt, voice input
should work across more models, and MCP servers should recover more reliably
after authentication changes.

## Highlights

- **App Sidebar is more stable** with better close behavior, fewer unwanted
  reloads, and remembered width per app.
- **Screen Doodle is easier to reach** from SmartBar, App Sidebar, Chat Window,
  Dynamic Island, Session, and the slash command menu.
- **Default Agent switching is easier** from the agent icon in SmartBar, Chat
  Window, and the Dynamic Island panel.
- **Dynamic Island follow-up input is steadier** with better text growth,
  window sizing, and context-item handling.
- **Voice Command and Dictation are more reliable** with Dynamic Island hotkeys
  and local MLX-based speech models.
- **MCP servers recover better** when OAuth tokens expire or server credentials
  need to be refreshed.
- **Stopping chat now really stops the stream**, so cancelled conversations no
  longer keep using tokens in the background.

## App Sidebar

- Fixed App Sidebar close behavior when clicking the close button, closing the
  tracked app window, or restoring an app after minimize.
- Kept the App Sidebar WebView alive across close and reopen flows, so reopening
  the sidebar no longer reloads the whole panel unnecessarily.
- Added per-app width memory, so each application can keep its own sidebar
  width.
- Improved the tether indicator when re-enabling App Sidebar from the menu bar.
- The tether indicator now hides when an app icon is unavailable instead of
  falling back to the default Enconvo icon.

## Chat, Agents, And Screen Doodle

- Improved chat scrolling during streaming and long conversations.
- Fixed user messages not scrolling to the expected position in long chat
  histories.
- Fixed stopping a chat session so the underlying LLM request is cancelled
  immediately.
- Added Default Agent switching from the agent icon in SmartBar, Chat Window,
  and the Dynamic Island panel.
- Added Screen Doodle to the attachment menu across SmartBar, App Sidebar, Chat
  Window, Dynamic Island, and Session.
- Added Screen Doodle to the slash command menu.

## Voice And Dictation

- Fixed Voice Command from the Dynamic Island hotkey with non-realtime speech
  models such as Whisper, Parakeet, and Qwen ASR.
- Improved Dynamic Island follow-up input so longer text grows naturally without
  overtaking the panel.
- Fixed Dynamic Island window clipping when context items expand the input area.
- Fixed download, delete, and load actions for local MLX-based Dictation models
  such as Qwen ASR, Parakeet, and Whisper.
- Reordered Dictation & Transcription settings so Dictation appears before
  Dictation Models.
- Fixed the Dynamic Island follow-up input animation when adding context items.

## MCP And Runtime Reliability

- Added clearer MCP server install feedback in Settings.
- Fixed MCP OAuth servers losing connection after access tokens expired.
- Fixed expired MCP OAuth servers showing an already-authorized message instead
  of reconnecting.
- Fixed MCP OAuth credential refresh when a server registration changes.
- Fixed RTK setup and rewrite handling so supported commands can run with much
  lower token usage.

## Settings And Model Polish

- Fixed a Settings page flash during load.
- Updated Xiaomi MiMo icons across credentials, chat models, and TTS.
