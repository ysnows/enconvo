# Enconvo 2.5.5 Beta 8

Thanks for testing Enconvo 2.5.5 Beta 8. This beta is still being prepared.

## Highlights

- **Faster, calmer startup** — the extension runtime no longer rewrites
  hundreds of files, re-installs Python packages, or launches every startup
  task at once on each launch.

## Startup Performance

- Fixed the extension runtime rewriting hundreds of installed-command files on
  every launch; unchanged commands are now left untouched, cutting startup disk
  and CPU churn.
- Removed the background file watchers on installed commands and extensions,
  whose change storms caused repeated re-parsing, database writes, and UI
  refreshes after startup.
- Changed app-startup extension hooks to run a couple at a time instead of all
  seventeen at once, smoothing the CPU spike right after launch.
- Changed the yt-dlp updater to check for updates at most once per day in the
  background, instead of re-resolving its Python environment on every launch.
- Changed the MLX Python environment to skip its dependency sync when nothing
  changed since the last successful sync.
- Changed the ripgrep, RTK, and caesiumclt auto-installers to wait a day before
  retrying a failed download instead of retrying on every launch.
- Added a startup performance log that records how long each launch phase and
  extension startup hook takes, so slow starts can be diagnosed from a file.

## Models

- Fixed LM Studio missing from the Smart Bar and Chat Window model pickers — it
  now lists the models served by your local LM Studio server (with context
  length and vision detection) instead of borrowing OpenAI's model list, and it
  appears once the LM Studio server has been detected.
- Fixed models whose reasoning profile uses a "disabled" effort level (DeepSeek
  V4, GLM 5.2, Kimi) failing on OpenAI-compatible endpoints — the setting now
  correctly omits the reasoning parameter.

## Fixes

- Fixed the onboarding guide window washing out to gray when macOS is in light
  mode — its frosted glass now stays dark to match the guide's design.
- Changed the free plan to include one knowledge base and one workflow; creating
  additional ones now requires an Enconvo Cloud plan or a lifetime license.
