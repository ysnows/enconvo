# Enconvo 2.5.5 Beta 17

Thanks for testing Enconvo 2.5.5 Beta 17. This build focuses on permissions,
workflow editing, faster panels, and a more useful Pet while tightening several
voice, capture, and automation flows.

## Highlights

- **Permissions are easier to set up** with a new Settings page and onboarding
  step for Accessibility, Screen Recording, Input Monitoring, Microphone,
  Notifications, Camera, Calendar, Reminders, Location, and Automation.
- **Workflow editing now has a visual builder** with a node canvas, schema-based
  step forms, live run status, per-step execution, and a cleaner YAML fallback.
- **SmartBar and panel interactions are faster** when opening, typing, searching,
  sending long responses, and launching apps from results.
- **Pet now shows live task cards** with current tool activity, finished-answer
  previews, quick jump/reply actions, and custom pet pack support.
- **Voice, screenshots, and read-aloud are more reliable** across Dynamic Island,
  SmartBar, meeting recording, captioning, and TTS playback.

## Permissions

- Added a dedicated Permissions page in Settings with live status and Grant
  actions for all macOS permissions Enconvo uses.
- Added an onboarding Permissions step so new users can finish system setup in
  one place instead of finding separate privacy prompts later.
- Fixed screenshot, meeting recording, live captioning, camera, notification,
  and input-monitoring flows so missing permissions show the right prompt or
  System Settings page instead of silently failing.
- Changed startup behavior so camera, microphone, and Screen Recording prompts
  appear only when the related feature is used.

## Workflow Builder

- Added a visual workflow editor with step nodes, parallel job groups, insert
  points, an Inputs node, and a searchable catalog for built-in steps, installed
  plugins, and connected MCP tools.
- Added a step inspector that builds parameter forms from the selected tool's
  schema, keeps a raw JSON fallback, and shows the latest output and errors.
- Added single-step runs from the canvas and detail panel, including upstream
  dependency execution and live data-flow animation.
- Changed YAML mode into a focused full-width editor with diagnostics, run
  inputs, run history, and results shown as slim supporting panels.
- Fixed saved workflow and subworkflow execution errors caused by re-validating
  already-normalized workflow definitions.

## SmartBar And Panels

- Changed SmartBar search to avoid repeated frontmost-app snapshots, repeated
  backend searches, and large context re-serialization while typing.
- Changed the first SmartBar keystroke to search immediately while coalescing
  rapid follow-up keystrokes.
- Changed long streaming answers to append text in place so large replies stay
  smooth as they grow.
- Fixed the empty SmartBar composer choosing the wrong command when the default
  agent was only automatically highlighted.
- Changed launching apps from SmartBar results to use the native launcher, clear
  the input, and dismiss the bar without creating a chat message.
- Fixed Dynamic Island screenshot actions so Screenshot Explain and Screenshot
  Translate continue inside the island instead of pulling SmartBar forward.
- Fixed the Dynamic Island follow-up input size and paste behavior after
  screenshot and text-paste workflows.

## Pet

- Changed the Pet toggle to a clearer Wake Pet / Put Pet Away control.
- Added custom pet pack support from the My Pets page, with an Open Folder
  shortcut for the local pets directory.
- Added live task cards under the Pet showing current tool activity, waiting
  state, finished answer previews, jump/reply actions, and dismiss controls.
- Fixed App Sidebar sessions opened from the Pet so the correct app activates
  before its sidebar panel is shown.

## Voice, Audio, And Capture

- Changed Dictation, Recording, Transcription, and Captioning model lists to
  render immediately while local model download status fills in in the
  background.
- Fixed slow or stuck Dictation Models loading when the local MLX runtime was
  cold or failed to initialize.
- Fixed failed MLX Python startups leaving orphaned processes that could make
  later startups keep failing.
- Fixed Gemini TTS and Enconvo Cloud Gemini TTS producing no audio by writing
  generated WAV files atomically.
- Fixed read-aloud playback so switching TTS providers cannot leave two voices
  playing at the same time.
- Added Computer Use screenshots for hidden, minimized, off-Space, or background
  windows through ScreenCaptureKit.
- Fixed Computer Use permission errors so closed, minimized, hidden, or off-Space
  target windows no longer report a misleading Screen Recording issue.

## Knowledge Base, Automation, And Account

- Changed Knowledge Base folder and Add menus to use clearer labels such as
  Import Existing Folder.
- Changed Enconvo Cloud Mistral OCR so it no longer consumes points.
- Fixed cron jobs that failed to schedule, overlapped manual runs, or stayed in
  a permanent Running state after app restart.
- Changed cron Run Now results and scheduled-job API responses to report the
  real run outcome, parsed parameters, and live schedule status.
- Fixed product-update email preferences and secure unsubscribe links by syncing
  account choices and Resend delivery events.
