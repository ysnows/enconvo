# Enconvo 2.5.5 Beta 11

Thanks for testing Enconvo 2.5.5 Beta 11. This build focuses on meeting
recording, local transcription models, custom AI providers, Knowledge Base
behavior, and clearer account usage pages.

## Highlights

- **Meeting recording is easier to review** with speaker diarization, speaker
  recognition, and word-level alignment during audio playback.
- **Local transcription models are better supported** for recordings and
  transcription settings, so more speech work can stay on device.
- **Custom LLM providers are more reliable** with provider-specific model lists,
  saved model selections, and consistent model pickers across Chat and Smart Bar.
- **Knowledge Base usage is more consistent** when attached or @-mentioned
  knowledge bases are used in Chat and Agent mode.
- **Account and Points Usage pages are clearer** with improved plan status,
  membership display, and usage history logic.

## Meeting Recording

- Added speaker-labeled transcripts for multi-speaker recordings, so turns are
  grouped by speaker with clear labels.
- Added automatic speaker recognition from a personal voiceprint library.
- Added word-by-word transcript highlighting synced to audio playback, with
  click-any-word playback.
- Changed recording transcripts to keep punctuation, support copying, and split
  long single-speaker recordings into readable sections.
- Fixed recording durations and audio-player layout issues in the recording
  view.

## Local Transcription Models

- Changed transcription settings to separate Dictation and Transcription model
  choices more clearly.
- Changed the Transcription Models list to show Soniox first, then on-device
  local models.
- Added on-device voice-activity detection so silent clips are skipped instead
  of becoming stray text.

## Custom LLM Providers

- Fixed custom OpenAI-compatible and Anthropic-compatible providers so each one
  loads models from its own endpoint and credentials.
- Fixed selected custom-provider models not sticking in new chats.
- Changed the Chat window and Smart Bar model pickers to use the same provider
  and model loading behavior.

## Knowledge Base

- Fixed attached knowledge bases not being searched in Agent mode.
- Changed @-mentioned knowledge bases so their content is retrieved
  consistently in both Chat and Agent mode.

## Account And Usage

- Added clearer Account page sections for Lifetime licenses and Cloud plans.
- Changed membership names and plan display logic so the current account status
  updates immediately after purchase, refund, or plan changes.
- Changed Points Usage history to show real activity more clearly and hide
  internal reconciliation entries.
- Added an Upgrade shortcut in the settings sidebar for users on the free plan.
