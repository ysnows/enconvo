# Enconvo 2.5.5 Beta 24

Thanks for testing Enconvo 2.5.5 Beta 24. This build focuses on making Voice
Command feel immediate, keeping Dynamic Island transitions smooth, and improving
agent tool execution.

## Highlights

- **Voice Command starts recording immediately** when the hotkey goes down, so
  the first words are captured while the HUD animation is still opening.
- **The Dynamic Island voice HUD now has a live waveform** for realtime and
  standard dictation, including an opening wave animation and a processing wave
  while standard dictation transcribes.
- **Dynamic Island transitions are smoother** because unrelated settings writes,
  update checks, and compact-window shrinking no longer land on the opening
  animation frames.
- **AI agents handle multiple tool calls more reliably** across Anthropic,
  OpenAI, and Google Gemini providers.
- **Inline Mode looks lighter** with a smaller single-line indicator, shimmered
  command text, an aurora edge glow, and a hover-to-cancel progress arc.

## Voice Command

- Changed the voice command hotkey to start recording on key-down, with a second
  tap stopping just as quickly.
- Changed the voice command session so the microphone begins capturing
  immediately while transcription setup and app-context collection wait until
  the opening animation lands.
- Fixed remaining opening stutter by moving start-sound decoding, system-volume
  mute/restore, and microphone device lookup off the main thread.
- Fixed Escape during a voice command session so canceling the HUD no longer
  passes the key press through to the app you are working in.
- Fixed an app-wide freeze that could happen when starting Voice Command a
  second time.

## Dynamic Island HUD

- Added a live waveform to the voice command HUD, replacing the waiting
  placeholder before words arrive.
- Changed realtime and standard dictation to share the same waveform experience
  in the voice command HUD.
- Added a center-out opening animation for the waveform.
- Changed the waveform to stand taller, respond better to normal speaking
  volume, and keep a gentle resting ripple during silence.
- Changed the HUD layout so the status label sits under the physical notch on
  notched displays and centers in the menu bar band on other displays.
- Changed the compact Dynamic Island pill to show Working immediately after a
  voice-command submit.

## Reliability And Agents

- Changed Dynamic Island transitions so app-wide settings writes no longer
  trigger unrelated island re-renders.
- Changed compact Dynamic Island shrinking to wait for outgoing content before
  resizing the window.
- Changed analytics and update checks to run after the expansion animation has
  settled.
- Fixed the SmartBar command list hover "..." button so a left click opens the
  row actions menu instead of executing the command.
- Fixed multi-tool turns for AI agents so multiple tool calls in one model turn
  each run and report their own result.

## Inline Mode

- Changed the Inline Mode indicator to a compact single-line pill.
- Changed the progress treatment to use shimmered command text, an aurora bottom
  glow, and a gradient comet arc that becomes the cancel button on hover.
