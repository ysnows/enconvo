# Enconvo 2.5.5 Beta 16

Thanks for testing Enconvo 2.5.5 Beta 16. This build focuses on the new Pet
experience, a much richer Knowledge Base, faster panel launches, and more
reliable voice and Read Aloud behavior.

## Highlights

- **The Pet replaces the Companion Orb** with an animated desktop companion
  that reflects agent activity, supports community pet packs, and can open your
  preferred Enconvo surface.
- **Knowledge Base now feels like a real asset browser** with a folder tree,
  preview grid, inspector, tags, drag and drop, paste import, media previews,
  and a standalone Knowledge Base window.
- **Quick Capture and Voice Note are faster and more useful**: captures appear
  almost immediately, recordings save instantly, and both can target any
  knowledge base.
- **Panels open faster from hotkeys** across SmartBar, small SmartBar, Chat
  Window, and App Sidebar, with context loading moved into the background.
- **Voice input and Read Aloud are more reliable**: dictation now merges with
  typed edits instead of replacing them, and audio playback failures now show
  clear errors.

## Pet

- Added the Pet, an animated desktop pixel companion with Boba as the built-in
  default.
- Added a Pet gallery in Settings for browsing, filtering, previewing, and
  installing community pets from petdex.dev.
- Added Pet interactions for hover, drag, fling, and session-aware tapping.
- Changed the old Companion Orb into the Pet and retired the floating "Ask
  anything" orb pill.
- Fixed Pet animation and install edge cases, including spritesheets with fewer
  frames and packs whose display name differs from their pack id.

## Knowledge Base

- Added an Eagle-style Knowledge Base browser with folders, item counts,
  previews, grid and list views, marquee selection, right-click menus, and an
  inspector.
- Added a standalone Knowledge Base window with Chat Window-style chrome,
  navigation controls, a menu entry, and the global Option-Command-K shortcut.
- Added folder organization with Finder drag and drop, paste-to-import, item
  moves, tags, type filters, and search.
- Added richer previews for audio, video, images, PDFs, EPUBs, office files,
  markdown, YouTube captures, Quick Capture, and Voice Note items.
- Added per-stage pipeline status and automatic recovery for interrupted
  indexing work.
- Changed Quick Capture to store app content as a clean readable outline, and
  changed YouTube captures to save subtitles without surrounding UI clutter.
- Fixed duplicate indexing, stale moved-file paths, broken item identities,
  slow list loading, failed capture placeholders, thumbnail loss, and transcript
  edits racing with reindexing.

## Capture And Notes

- Changed Quick Memory to Quick Capture across the app.
- Added Dynamic Island buttons for Quick Capture and Voice Note.
- Added a "Save to" knowledge base picker for Quick Capture and Voice Note.
- Changed Voice Note capture to save audio immediately while transcription and
  indexing continue in the background.
- Added a dedicated Voice Note preview with an editable transcript, one-click
  re-transcription, and a Recordings-style player.

## Panels And Context

- Changed SmartBar, small SmartBar, Chat Window, and App Sidebar hotkey launches
  to show the panel immediately while reset and context attachment continue in
  the background.
- Changed Context Awareness to a single Selected Text switch, while manual
  context attachments remain available.
- Removed automatic Context Awareness suggestion rows from the SmartBar command
  list.
- Fixed selection-based hotkeys preserving selected text after instant panel
  launch.
- Fixed background commands opening the Chat Window unexpectedly.
- Changed Screenshot Explain and Screenshot Translate to capture first and open
  directly on the answer.

## Voice And Audio

- Fixed SmartBar and Dynamic Island voice input replacing text typed or edited
  during dictation.
- Changed voice input joining to add spaces between English words without
  inserting spaces inside Chinese, Japanese, or Korean text.
- Changed the voice input Cancel button to remove only dictated text.
- Fixed non-realtime dictation and Voice Trigger hotkey flows so existing typed
  text is preserved.
- Fixed Read Aloud error handling, silent failures, stale playback sessions,
  empty speakable text, unrelated playback state changes, and empty cached audio
  files.
- Fixed macOS System voice playback so the selected voice and default voice
  configuration produce sound correctly.

## Plugins And Account

- Added Claude Code and Codex plugin compatibility for `.claude-plugin` and
  `.codex-plugin` manifests.
- Added plugin support for root `.mcp.json` files, bundled MCP server variable
  expansion, and Claude Code/Codex skill layouts.
- Fixed uninstalling plugins so registered skills are removed from search.
- Added product-update email preferences during registration and in Account
  settings, plus secure unsubscribe links for product update emails.
