# Enconvo 2.5.5 Beta 23

Thanks for testing Enconvo 2.5.5 Beta 23. This build focuses on faster inline
commands, a simpler PopBar setup, richer app-aware results, and a more polished
Dynamic Island voice experience.

## Highlights

- **Inline Mode runs commands without opening a result window**, using a compact
  Liquid Glass indicator near the selected text and pasting the result back in
  place.
- **PopBar is easier to configure** with one command list, quicker command
  creation, in-place editing, and fewer legacy options.
- **Weather answers can now appear as native chat cards** with current
  conditions, hourly details, and multi-day forecasts.
- **Dynamic Island Voice Command has a new glass HUD** with live transcript,
  voice-responsive glow, and clearer running status.
- **More media providers are available** with Google Gemini video generation
  and Kie.ai image/video generation.

## Inline Commands And PopBar

- Added Inline Mode for PopBar and hotkey commands, including live status,
  hover-to-cancel, background paste-back, and automatic full-field selection
  when the caret is inside a text field.
- Changed PopBar settings to a single configuration where the command list is
  edited directly on the page.
- Added quick prompt-command creation and in-place command editing from PopBar
  settings.
- Removed older PopBar options such as auto-replace, legacy inline mode, long
  right-click launch, and last dragged position.
- Fixed PopBar hotkeys feeling delayed by removing duplicate selected-text
  capture.

## SmartBar And Command Settings

- Added command favorites in SmartBar so frequently used commands stay at the
  top of the command list.
- Fixed SmartBar row hover, row reuse, unrelated re-renders, and the row action
  button behavior.
- Changed the command picker to prioritize AI Commands and writing actions, and
  fixed older search responses overwriting newer results.
- Added a variables and template-syntax reference to variable-enabled prompt
  fields.
- Fixed command settings controls visually snapping back after changes were
  saved.
- Changed credential provider settings to show available sign-in methods inline
  when more than one option is available.

## Weather, Media, And App Sidebar

- Added a Weather plugin with current-location and local-weather APIs for
  current conditions, AQI, hourly forecasts, and daily forecasts.
- Added a native weather card for chat and delivered views, including city
  queries in multiple languages.
- Added Google Gemini video generation providers for your own API key or the
  Enconvo Cloud plan.
- Added Kie.ai image and video generation providers with curated model families,
  local media upload, and asynchronous task completion.
- Changed App Sidebar chats to run in agent mode when the selected model
  supports tool use.
- Added the App Sidebar launch film to the enconvo.com homepage hero showcase.

## Dynamic Island And Reliability

- Changed Dynamic Island Voice Command to open a half-expanded glass HUD with
  live transcript, voice level feedback, and compact progress status.
- Changed permission requests to use one unified flow across macOS permission
  dialogs and Enconvo's permission window.
- Fixed the permission guide briefly flashing before System Settings opened.
- Fixed Computer Use clicks inside Enconvo's own windows by falling back to real
  mouse events.
- Fixed draggable window areas showing the wrong cursor during native dragging.
- Fixed YouTube videos in the Knowledge Base preview showing "Error 153" instead
  of a playable player.
