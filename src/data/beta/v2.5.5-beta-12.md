# Enconvo 2.5.5 Beta 12

Thanks for testing Enconvo 2.5.5 Beta 12. This build focuses on live
captioning, richer App Sidebar capabilities, more reliable browser and computer
automation, and smoother model and speech workflows.

## Highlights

- **Live Captioning is now a full workflow** with realtime captions, speaker
  labels, local models, live translation, and saved captioning history.
- **App Sidebar can bring app-specific capabilities** such as skills and MCP
  servers into conversations for apps and websites.
- **Browser Use and Computer Use are more reliable** when tracking opened tabs,
  changed windows, app focus changes, and precise click targets.
- **GPT-5.6 is now available** through ChatGPT subscriptions, your own API key,
  or the Enconvo Cloud Plan.
- **Grok 4.5 and Grok Build capabilities are now available** with support for
  xAI models, voice input, text-to-speech, image generation, video generation,
  and search.
- **Model pickers and custom providers are easier to use** with instant fuzzy
  search, provider-aware results, and more reliable custom provider model lists.
- **Agent terminal and voice playback are more robust** for long-running shell
  commands, interactive CLIs, mixed-language speech, and stalled TTS providers.

## Live Captioning And Speech

- Added Live Captioning with realtime caption bubbles, speaker labels, saved
  Captioning history, and Dynamic Island shortcuts.
- Added live translation controls for supported realtime models, including
  original and translated text in each caption bubble.
- Added local realtime model support with paragraph segmentation, offline
  speaker detection, voiceprint naming, and crash recovery for interrupted
  sessions.
- Changed Recording, Transcription, Captioning, and Dictation to use separate
  model choices so each workflow keeps the right model.
- Added an audio source panel for choosing microphone, system audio, app audio,
  language options, and the model before starting.
- Improved the Dictionary and speech settings pages for larger dictionaries and
  clearer speech asset management.

## App Sidebar

- Added per-app capability profiles for Applications and Websites, including
  preset skills, MCP servers, instructions, and setup guidance.
- Added curated presets for Apple apps, browsers, Notion, Blender, JetBrains
  IDEs, Affinity, Obsidian, and other app-specific workflows.
- Changed App Sidebar messages to include a compact app context and screenshot,
  reducing token usage while still giving the agent enough context.
- Added capability badges, one-click setup, and in-place popovers so app
  capabilities are easier to review and customize.
- Fixed App Sidebar setup states, preset MCP authorization flows, and app-bound
  conversation context so pinned capabilities are applied consistently.
- Fixed switching to a session from the App Sidebar task list showing no
  messages when more than one App Sidebar panel was open; the switch could land
  in another app's sidebar instead of the one you clicked in.

## Automation Reliability

- Added same-window tab tracking and better window-change reporting for browser
  and computer automation.
- Fixed browser automation click targeting in list-style dialogs where tiny text
  fragments could send the click to the wrong row.
- Added frontmost-app switch notices when an action moves focus to another app.
- Changed browser-use and computer-use guidance so browser content and native UI
  control route to the right tool with graceful fallback.

## Models And Providers

- Fixed newly created custom providers missing from the Smart Bar and Chat
  window model pickers.
- Changed model picker search to fuzzy match instantly across warmed provider
  catalogs.
- Added GPT-5.6 support through ChatGPT subscriptions, your own API key, or the
  Enconvo Cloud Plan.
- Added Grok 4.5 support through Grok Build subscriptions, xAI subscriptions,
  the Enconvo Cloud Plan, or your own API key.
- Added Grok Build access for xAI voice input, text-to-speech, image generation,
  video generation, and search.
- Fixed provider-specific model selection issues across Chat, Smart Bar, and
  Dynamic Island.

## Agent And UI Polish

- Changed long-running shell commands to run in a real terminal, hand control
  back to the agent quickly, and support interactive prompts.
- Fixed Agent voice playback skipping, repeating, or stalling on mixed-language
  responses and rate-limited TTS providers.
- Fixed SmartBar and PopBar focus behavior so source-app text selections stay
  active.
- Improved settings dialogs, skill management, speech Dictionary selection, and
  Dynamic Island model search focus.
