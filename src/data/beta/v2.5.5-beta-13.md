# Enconvo 2.5.5 Beta 13

Thanks for testing Enconvo 2.5.5 Beta 13. This build focuses on making app
automation setup clearer and keeping provider choices closer to the services
that actually power each workflow.

## Highlights

- **Sketch gets a dedicated App Sidebar preset** for design automation through
  Sketch's built-in MCP server.
- **Video Generation provider choices are cleaner** with unavailable Enconvo
  Cloud Plan and OpenAI options removed from the default list.
- **Gmail now starts with your own Google OAuth credentials by default** while
  the Gmail (Enconvo Cloud) option remains available when you want it.

## App Sidebar

- Added a Sketch preset that pins Sketch's built-in MCP server for design
  automation.
- Added setup guidance that walks through starting Sketch's MCP Server when it
  is not already running.
- Added Sketch automation support for running SketchAPI code, inspecting
  documents, reading layers and symbols, and exporting assets.

## Providers

- Removed unavailable Enconvo Cloud Plan and OpenAI choices from the Video
  Generation provider list.
- Changed Gmail's default credential provider to your own Google OAuth
  credentials.
- Kept Gmail (Enconvo Cloud) available as a manual credential choice.
