# Enconvo 2.5.5 Beta 10

Thanks for testing Enconvo 2.5.5 Beta 10. This build focuses on faster local
dictation, lower CPU usage, better prompt caching, and clearer usage tracking.

## Highlights

- **New CoreML dictation models** now run fully on device with smaller downloads,
  lower memory use, and faster Apple Neural Engine transcription.
- **Prompt cache hit rate is now around 98%** for supported conversations, so
  repeated context is reused more reliably.
- **High CPU usage is fixed** by reducing unnecessary background work after the
  app starts and while chat surfaces are open.
- **A new Points Usage page** makes it easier to understand recent usage and
  cost patterns.

## CoreML Dictation

- Added CoreML dictation support for Parakeet, SenseVoice, Paraformer, and
  Nemotron 3.5 Multilingual models.
- Parakeet is optimized for fast multilingual, English, and Japanese
  transcription.
- SenseVoice works well for Chinese, Cantonese, English, Japanese, and Korean.
- Paraformer focuses on Mandarin Chinese transcription.
- Nemotron 3.5 Multilingual supports realtime dictation across about 40
  languages, with live partial text while you speak.
- Changed local CoreML model handling so downloads are smaller, memory use is
  lower, and transcription starts faster when a model is already downloaded.

## Performance

- Changed conversation context handling to keep prompt-cache-friendly content
  stable between turns, improving cache hits to about 98% in supported flows.
- Fixed high CPU usage caused by repeated background work and retained chat
  surfaces.

## Usage

- Added a new Points Usage statistics page for reviewing recent usage more
  clearly.
